## 💡 为什么要用 FourThings？

市面上的 Todo 应用多如牛毛，FourThings 有何不同？

它不仅仅是一个清单，它是一个**视觉化的决策面板**。它强迫你在创建任务的那一刻就思考：**这件事现在必须做吗？它重要吗？**

通过将任务划分为四个象限，FourThings 让你一目了然：
1.  🔴 **重要且紧急**：立即去做！
2.  🟡 **重要不紧急**：制定计划，这是你成长的关键。
3.  🔵 **紧急不重要**：学会拒绝或授权他人。
4.  ⚪ **不紧急不重要**：尽量少做，甚至不做。

## ✨ 核心亮点

### 1. 🎯 极致直观的四象限视图
告别枯燥的列表！FourThings 提供了一个清晰、美观的四象限看板。
*   **拖拽交互**：任务优先级变了？直接拖拽到另一个象限，流畅丝滑。
*   **独立滚动**：每个象限独立管理，任务再多也不乱。
*   **视觉反馈**：通过颜色和布局，让你一眼识别当务之急。

### 2. 🛡️ Local-First (本地优先)，数据完全掌控
在这个数据泄露频发的年代，我们深知隐私的可贵。
*   **完全离线可用**：基于 **IndexedDB** 技术，你的数据首先存储在你的浏览器本地。无论是否有网络，都能流畅使用。
*   **隐私安全**：除你之外，没人能通过后台偷窥你的计划。

### 3. ☁️ 无缝云端同步 (可选)
想要多设备协作？没问题！
*   **智能同步**：登录后，利用 **Vercel Serverless** 和 **Postgres** 强大的后端，数据将在数秒内自动备份到云端。
*   **多端一致**：在电脑上规划，在手机上查看（PWA 支持即将完善），随时随地保持同步。

### 4. 📊 数据可视化 (ECharts 驱动)
不仅仅是记录，更是复盘。
*   通过内置的图表功能，回顾你的时间都花在了哪个象限。你是“救火队员”（总在做重要紧急），还是“长期主义者”（专注于重要不紧急）？数据会告诉你答案。
